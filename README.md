# 📊 Amazon Sales Analysis Dashboard (Near Real-Time)

An end-to-end data analysis project built using **Excel, SQL, and Power BI** to analyze Amazon sales data. This dashboard provides actionable insights into revenue trends, customer behavior, and product performance using a dataset of **100K+ orders**.

---

## 🚀 Project Overview

The goal of this project is to:
- Analyze large-scale sales data
- Track key business KPIs
- Identify revenue patterns and customer trends
- Enable data-driven decision-making

---

## 🛠️ Tools & Technologies

- **SQL** – Data querying and analysis  
- **Microsoft Excel** – Data cleaning and preprocessing  
- **Power BI** – Interactive dashboard and visualization  

---

## 📈 Key Features

- Tracks KPIs such as:
  - **Total Revenue ($91M+)**
  - **Average Order Value**
  - **Order Volume**
- Visualizes:
  - Sales by **Country, Category, and Product**
  - **Payment Method distribution**
  - **Order Status analysis**
- Identifies:
  - Top-performing regions (**USA**)  
  - Best-selling products  
- Displays:
  - Monthly revenue trends  
- Supports **near real-time updates** using data refresh  

---

## 🧠 Insights Generated

- USA contributes the highest share of revenue  
- Certain categories dominate overall sales  
- Majority of orders are successfully delivered  
- Digital payment methods are widely used
